## Key Concepts

### LangChain vs Open Router
- **Open Router** — a platform that *hosts* models. You generate an API key and call their models.
- **LangChain** — a framework that gives you *integration*. It wraps all providers (OpenAI, Groq, Gemini etc.) in the same interface. You bring your own API key.

### Prompt Structure
Every message sent to an LLM has two parts:
- **System message** — sets permanent behavior of the model for the current session (e.g. "You are a helpful assistant")
- **Human message** — the actual user input/question
- **AI message** — the output generated by the model

### Training vs Inference
- **Training** — processes all tokens in parallel. Fast because the full answer already exists in training data.
- **Inference** — input is read in parallel, but output is generated **sequentially**, one token at a time (T=1, T=2, T=3…). Each token depends on all previous tokens.

### Base64 Encoding
Images and audio cannot be passed directly to a model. They must be converted to binary format first using Base64 encoding. This applies to images, audio files, and video.

### Tokens
A token ≈ a word or word-fragment. Both input and output are measured in tokens. Free API tiers have hard token limits (e.g. Open Router free ≈ 2659 tokens combined).

## The 4-Step Invoke Pattern

This pattern is identical across all providers — only the import and model name changes.

```python
# Step 1 — set API key
import os
os.environ["PROVIDER_API_KEY"] = "your-key"

# Step 2 — initialize model
from langchain_xyz import ChatXYZ
llm = ChatXYZ(model="model-name", max_tokens=1024)

# Step 3 — define message
messages = [
    ("system", "You are a helpful assistant"),
    ("human", "Your question here")
]

# Step 4 — invoke
response = llm.invoke(messages)
print(response.content)
```

## Multimodal Models

| Modality | LangChain Wrapper? | Notes |
|---|---|---|
| Text → Text | Yes | Core use case |
| Image → Text | Yes | Requires base64 encoding. Used for OCR via Vision Transformer (ViT) |
| Audio → Text | Yes (Gemini) | Encode audio as base64, pass MIME type |
| Text → Audio | No | Use Google Gen AI SDK directly. Voice persona configurable |
| Text → Image | Partial | Provider-specific |

> When no LangChain wrapper exists, use the provider's native SDK directly (Google Gen AI SDK, OpenAI SDK etc.)
